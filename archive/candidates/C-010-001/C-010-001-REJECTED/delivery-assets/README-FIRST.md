# Handyman Quote Readiness Planner

Open `Handyman-Quote-Readiness-Planner.xlsx` first.

1. Confirm your defaults in `Assumptions`.
2. Complete the readiness inputs in `Scope Inputs`.
3. Enter task rows only in yellow cells.
4. Read `Quote Summary` and follow the next-step recommendation.

Use `Saved Quotes` only to preserve snapshots. This product does not invoice, email clients, or schedule crews.
