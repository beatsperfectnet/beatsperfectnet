# Meal Planner Studio

Open `Meal-Planner-Studio.xlsx`, then start on the `Start Here` tab. Keep one untouched backup copy before editing.

First success:
1. Review the sample Dashboard.
2. Edit calorie target and food budget in Settings.
3. Edit Pantry and Recipes.
4. Choose meals in Weekly Planner.
5. Use Grocery List for shopping. The grocery list is driven from the weekly plan and recipe ingredients.
