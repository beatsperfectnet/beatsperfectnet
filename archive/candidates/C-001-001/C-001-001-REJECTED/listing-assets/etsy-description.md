Plan a week of meals, build reusable recipes, and generate a practical grocery list from one calm spreadsheet workbook. Meal Planner Studio is built for busy households that want calories, estimated food cost, pantry context, and shopping prep in one place without a subscription.

What is included: Dashboard, Start Here, Settings, Recipes, Pantry, Weekly Planner, Grocery List, and Help. Each worksheet includes local instructions and row/column guidance. Editable areas are marked cyan, formula/output areas are protected, and the Grocery List uses a visible editable Unit price column instead of hiding prices inside formulas.

The workbook includes starter recipes, sample pantry data, editable setup fields, readable charts, dropdown meal choices, and a safe final grocery list. Use the provided cyan rows; inserting or deleting columns is unsupported because spreadsheet formulas depend on the shipped structure.

Google Sheets is the primary supported platform. Excel compatibility must pass final QA before it is claimed in the published listing. Nutrition and cost outputs are estimates only; this is not medical, diet, grocery-price, tax, or financial advice.
